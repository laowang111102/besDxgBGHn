# 学校运动会管理系统 python559

## 项目概述

学校运动会管理系统是一款基于Python和MySQL开发的软件，旨在提供运动会项目的报名、管理、成绩审核与查询等功能。系统分为管理员端、学生端和教师端，以满足不同角色的需求。

## 技术栈

- **后端语言**: Python
- **数据库**: MySQL
- **开发工具**: PyCharm
- **数据库管理**: Navicat for MySQL

## 功能模块

### 管理员端

- 项目管理
  - 添加、删除、修改和查询项目信息
- 成绩管理
  - 审核学生和教师成绩
  - 增删改查成绩数据

### 学生端

- 项目报名
- 成绩查询

### 教师端

- 项目报名
- 成绩提交

## 系统角色

- 管理员：负责运动会项目与成绩的管理
- 学生：可报名参加项目，查询个人成绩
- 教师：可报名作为裁判，提交成绩

## 运行环境

- 操作系统：兼容Windows、Linux、MacOS
- Python版本：3.x
- MySQL版本：5.x 或更高

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，配置数据库连接
3. 导入数据库文件
4. 部署应用程序到服务器
5. 运行应用程序
